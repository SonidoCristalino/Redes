Ejercicio del parcial nº 7:
--------------------------
Mandar un archivo de 5 millones de bytes, MTU 1500bytes, cabecera de IP 20.
(Si el ejercicio dice que el archivo "viene un segmento por internet"... es que ya tiene la cabecera).

De 1500 hay 20 bytes de cabecera y 1480 de cuerpo.

5000000 / 1480 = 3378 con un resto de 560(se debe sacar el resto, haciendo 5 millons - (3378*1480) )

    Se envian 3378 + 1 (el del resto):
        - se envian 3378 de 1500 bytes (porque se agregan los 20 de cabecera)
        - y se envía el último de 580: 560 + 20

Offset:
    1 - 0
    2 - 1480
    3 - 2960
    ...
    3379 - 3378

Resultado:
La cantidad de paquetes es de 3379 porque son los 3378 bytes de 1500 + 1 de 580

    1) sabe rqué router de borde se tiene que elegir para poder pasar a otra región autónoma.
    2) llegar a ese router de borde para que este alcance la otra región autónoma a donde se quiere llegar.

Uno es cuál debe ser el router de borde y el otro es cómo llegar hasta él.

¿Es necesario que todos los sistemas autónomos utilicen el mismo algoritmo de enrutamiento interno? ¿Por qué?
No es necesario, ya que cada región autónoma resuelve internamente el saber cuál es el ruter de borde y cómo llegar
hasta él. Se pueden llegar de distintas formas dependiendo del algoritmo, pero básicamente tienen que resolver los dos
íntems anteriores. (Verlo bien en el libro).
